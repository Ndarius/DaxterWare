Section Offline - Ajout manuel de setups

Placez vos installateurs ici, par classification :

offline/
  Bureautique/
    LibreOffice_7.6.4_Win_x86-64.msi
  Navigateurs/
    Firefox Setup 136.0.exe
  Utilitaires/
    7z2408-x64.exe

Règles :
- Les sous-dossiers (1er niveau) deviennent des catégories dans l'application : "Offline / <Classification>".
- Le nom affiché du logiciel est dérivé du nom du fichier setup.
- Extensions supportées : .exe, .msi, .msix, .zip
- Vous pouvez aussi mettre des setups directement à la racine de offline/ (classification "Général").
